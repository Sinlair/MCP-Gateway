MCP Gateway

令牌驱动的系统操作与监控入口

这里不是传统账号密码后台。你需要先由 MCP Gateway 签发访问令牌， 再使用这个令牌进入中文监控控制台。控制台会根据令牌自动决定： 当前可访问的系统、可见工具、可执行操作以及权限范围。

  • 网关负责：签发访问令牌、管理受管系统能力、治理工具暴露
  • 控制台负责：用 token 驱动系统操作、运行监控、策略与调用实验
  • 当前受管系统：`big-market-71772-z`
